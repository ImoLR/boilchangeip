# boilchangeip v2.1.0

本版本重点改进 Telegram Bot 的首次配对、服务器管理和状态展示体验。

## 新功能

- Telegram Bot 支持一次性配对码：`boil setup` 生成 `/pair CODE`，用户在 Bot 私聊发送后自动绑定当前 chat。
- `/start` 提供 Inline Keyboard 主菜单，不需要记忆常用命令。
- `/addserver` 提供交互式添加服务器向导，分步输入服务器名称、地址和 Token。
- `/servers` 支持服务器列表、编辑、删除和排序。
- 服务器编辑支持修改名称、地址、Token 和重新验证。
- 删除服务器前会二次确认，确认后同步删除该服务器自己的定时任务。
- `/status` 支持本地生成图片卡片，显示服务器名称、国家/地区、地址、状态和下次换 IP 倒计时。

## 改进

- Telegram 原生命令菜单包含 `/start`、`/status`、`/change`、`/timer`、`/servers`、`/addserver` 和 `/help`。
- 服务器展示统一为用户填写名称、国家/地区和地址，不显示内部 server id、Token 或旧 router/interface。
- 图片卡片生成失败或 Telegram `sendPhoto` 失败时自动回退为安全文本。
- Bot 代码拆分为命令、callback、配对、添加向导、编辑、删除、列表、格式化、状态和 Timer UI 等模块，便于后续维护。

## 兼容性

- 从 v2.0.2 升级无需手动迁移 `BOIL_SERVERS`。
- 新增的 `address`、`country`、`flag` 和 `resolved_ip` 字段均为可选字段。
- 已配置 `TG_CHAT_ID` 的用户无需重新配对。

## 安全

- 未绑定前只允许 `/start` 和 `/pair`。
- 已绑定后，非授权 chat 的消息和 callback 均会被拒绝。
- Token、配对码和服务器地址不会进入 Telegram callback。
- 修改 Token 会先验证，验证失败不会覆盖旧 Token。
